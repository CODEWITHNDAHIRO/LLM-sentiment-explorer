# LLM Sentiment Explorer

## Project Overview
This project explores sentiment analysis using Large Language Models (LLMs). We aim to compare traditional NLP preprocessing techniques with zero-shot and few-shot prompting using modern APIs.

## The Approach
1. **Data Collection**: Utilizing public sentiment datasets (e.g., IMDB or Twitter).
2. **Preprocessing**: Cleaning noise, handling tokenization, and formatting for LLM consumption.
3. **Evaluation**: Comparing LLM outputs against ground-truth labels using Accuracy and F1-score.

## Tech Stack
* **Language**: Python 3.9+
* **Models**: HuggingFace Transformers, OpenAI/Claude API
* **Tools**: Pandas, Scikit-learn, GitHub Actions (for CI/CD)
